En la carpeta pozos, se encuentra una serie de archivos que contienen lo siguiente:

-Archivo de excel con nombre BASE DE DATOS DEPURADOS SACMEX(2017).
 Contiene la depuración de información de bases de datos de nivel piezométrico de la CDMX. 
El la primera pestaña por  nombre "BASE_DE_DATOS" contiene información sin depurar.
La segunda pestaña contiene los datos depurados del nivel piezométrico. 
La tercera y cuarta pestaña contiene la desviación estándar para cada carga hidráulica y 
abatimiento considerado como observación, los cuales fueron tomados en cuenta y calculados por Rosa Galán. 


-Archivo de excel "BASE_DE_DATOS_MODELO". 
Contiene la depuración de información piezométrica de la base de datos utilizada para el modelo, 
agregando la nueva nueva información del archivo "BASE DE DATOS DEPURADOS SACXMEX(2017). 
En la primera tiene por nombre "Datos_modelo contiene la información de observaciones utilizados para el modelo, 
realizado por Rosa Galan. En la pestaña "información adicional" contiene  informacion nueva de los pozos que se agregaron al modelo, 
pozos que  contiene una misma coordenada e información de los pozos que se eliminaron.

La tercera pestaña por nombre "DATOS_DEPURADOS_MODELO"  
Es la base de datos que  ya se encuentra  depurada y lista para utilizar en la corrida del modelo, 
cabe mencionar que se agregó informacion de observaciones de los años de 1968 a 2017 que fueron depurados en el archivo 
"BASE DE DATOS DEPURADOS SACMEX(2017).

Por último en la cuarta y quinta pestaña se incluye la desviación estandar para cada carga hidráulica y abatimiento considerado como observación, 
los cuales fueron tomados por el archivo "pozos_pesos"  calculados por  Rosa Galán. 
Como comentario final, los pozos nuevos  que se agregran para la corrida del modelo, 
se les asignó un valor de desviación estandar,  como el promedio de todos los pozos que contienen  
informacion ya asignada( desviación estandar) y  los pozos que se le agregarón nuevas observaciones, 
se les colocó el mismo valor de desviación estandar utilizados en el modelo.
